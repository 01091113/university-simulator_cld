■ 1. 주거 유형 선택 기능
----------------------------------------------------------------
  - 시작 화면에 '주거 유형' 선택 항목 추가
    • 🍳 자취   — 초기 돈 -8, 스트레스 +3 / 자취 전용 이벤트
    • 🛏️ 기숙사 — 초기 인간관계 +4 / 기숙사 전용 이벤트
    • 🚇 통학   — 초기 돈 +6, 체력 -3 / 통학 전용 이벤트
  - 게임 화면 상단에 주거 유형 배지(pill) 표시
  - 학년 종료 리포트에 주거 유형 표시

----------------------------------------------------------------
■ 2. 주거 유형별 이벤트 (events_housing.js)
----------------------------------------------------------------
  자취 이벤트 8종:
    냉장고에서 정체불명의 냄새가 난다
    냉장고가 텅 비었다
    이불 빨래를 해야 할 때가 됐다
    옷이 산처럼 쌓였다
    관리비 고지서가 왔다
    새벽 1시 갑자기 배가 고프다
    벌레가 나왔다
    배달이 잘못 왔다

  기숙사 이벤트 6종:
    룸메이트랑 생활 패턴이 충돌한다
    기숙사 귀가 시간을 넘겼다
    샤워실 줄이 너무 길다
    옆 방이 너무 시끄럽다
    냉장고 음식이 사라졌다
    새벽 3시 화재 대피 훈련

  통학 이벤트 6종:
    지하철이 미어터진다
    첫차를 놓쳤다
    부모님이 공부 얘기를 꺼냈다
    집에 귀가 시간이 있다
    통학 시간이 너무 아깝다
    친구가 자취 안 하냐고 물었다

  공통 이벤트 1종 + 이벤트 트리거:
    - 'rest' / 'sleep' 선택 시 40% 확률로 주거 이벤트 팝업
    - 일반 랜덤 이벤트에서도 30% 확률로 트리거
    - 중복 방지: housingEventDone 플래그로 같은 이벤트 재출현 억제

----------------------------------------------------------------
■ 3. 이벤트 결과 텍스트
----------------------------------------------------------------
  - 이벤트 선택 후 결과가 로그에 기록됩니다
  - 형식: "이벤트 설명 ▶ 선택 결과"
  - 주거 이벤트 전체, 특수 루트 이벤트, 연계 이벤트에 적용
  - 늦잠 이벤트 4개 선택지 모두 결과 텍스트 추가

----------------------------------------------------------------
■ 4. 특수 이벤트 루트 (SPECIAL_ROUTE_EVENTS)
----------------------------------------------------------------
  자동 트리거 조건:
  ┌──────────────────┬──────────────────────────────────┐
  │ 루트             │ 조건                              │
  ├──────────────────┼──────────────────────────────────┤
  │ 학점 회복 루트   │ 학점 ≤ 20 && 아직 시작 안 함     │
  │ SNS 부업 루트    │ 덕질 ≥ 50 && 인간관계 ≥ 60       │
  │ 연인 위기 루트   │ 연애중 && 스트레스 ≥ 75          │
  │ 취업 패닉 루트   │ 4학년 && 취업역량 ≤ 40           │
  └──────────────────┴──────────────────────────────────┘
  - 각 루트는 2단계 이벤트로 구성 (연속 체인)
  - specialFlags 로 중복 방지
  - 특수 루트 모달은 노란 배경(variant-special)으로 구분

----------------------------------------------------------------
■ 5. 글자 크기 상향
----------------------------------------------------------------
  기본 body 폰트: 17px → 18px
  이벤트 설명: font-size 20px (기존 18px)
  선택지 버튼: font-size 17px (기존 16px)
  모달 텍스트: font-size 18px
  상태 카드 헤더: 15px
  로그 텍스트: 15px
